Moving maze game

This game start with 4 players each in one corner of a maze where the maze size is specief before the game such as 3 x 3 grid. The goal is for each player to collect their tokens that are scattered around the maze then return to their starting corner. Each player also has to push a block into the maze which can cause the maze to change before they move.
